# Instacart_analysis
This project explores Instacart grocery shopping data to analyze customer purchasing behavior, reorder patterns, popular products, and shopping habits across different days and hours. The analysis was completed in Python with pandas using a Jupyter notebook.


## What I Did
* Loaded multiple datasets using pandas
* Inspected the first rows, columns, and dataset structure
* Cleaned and standardized data
* Checked for missing values and duplicates
* Merged related datasets for analysis
* Explored product popularity and reorder behavior
* Compared order activity by day of week and hour of day
* Analyzed department and aisle purchasing trends
* Generated business insights based on customer behavior


## Dataset Summary
The analysis uses multiple Instacart datasets containing customer orders, product information, aisles, departments, and reorder activity.


## Key Findings
* Bananas were among the most frequently ordered products
* Produce and dairy departments had the highest purchase volume
* Most orders were placed during daytime hours
* Reorder rates were high, indicating strong repeat buying behavior
* Customer purchasing patterns varied across weekdays and weekends
